# 类与委托

## 类继承

```js
function Foo() {}
Foo.prototype.c = 'c';

function Bar() {}
Bar.prototype = Object.create(Foo.prototype);

const bar = new Bar();
bar.c // "c"
```

## 委托

```js
const foo = {
  a: 1,
  b: true,
  c: 'c',
}

const bar = Object.create(foo);
Object.keys(bar); // []
bar.a // 1
```
