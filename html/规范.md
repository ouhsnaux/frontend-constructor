# HTML 规范

1. 指定 `HTML` 版本 `<!DOCTYPE html>`
2. 指定语言 `<html lang="zh-CN">`
3. 指定编码 `<meta charset="UTF-8">`
4. 指定 `title`
5. 根据需要编写 `keywords` 和 `description`
6. 标签，属性及大部分属性值小写，属性值加双引号
7. 使用 `2` 个空格缩进
8. 具有开始和结束标签的元素都要写上起止标签，空元素不加 `/` 字符
9. 文件名小写，`_` 连接。
10. 尽量不适用内联文件，除非特别小。
11. 非特殊情况，`js` 文件要么外链，要么放到 `body` 底部。
